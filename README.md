# Simple Softbody Ragdoll

A simple 2D softbody ragdoll simulation using Python.

## Features
- **Physics Fundamentals**: Includes vector operations, particles, springs, and constraints.
- **Soft Objects**: Supports ropes, chains, softbodies, ragdolls, and cloth.
- **Integration Methods**: Euler, Verlet, and Semi-Implicit Euler for time-stepping.
- **Visualization**: Pygame, debug, and Matplotlib renderers.
- **Demo Scenes**: Predefined scenes for testing and demonstration.

## Installation
1. Clone the repository.
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the main script to start the simulation:
```bash
python src/main.py
```
